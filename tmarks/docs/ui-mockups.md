### 4. 面包屑导航组件（Breadcrumb）

```
┌────────────────────────────────────────────────────┐

│  ┌──────────────────────────────────────────────┐  │
```

**元素：**
- **文件夹路径**：📁 图标 + 文件夹名称 + [×] 按钮

- **标签**：🏷️ 图标 + 标签名称 + [×] 按钮

- **统计信息**：找到 X 个书签
- **清空按钮**：一键清空所有筛选

**交互：**
- 点击文件夹名称：跳转到该文件夹
- 点击标签名称：取消选中该标签
- 点击 [×]：移除该筛选条件
- 点击"清空所有筛选"：回到全部书签视图

## 响应式断点

### 断点定义

- **xs**：< 640px（小型手机）
- **sm**：640px - 768px（大型手机）
- **md**：768px - 1024px（平板）
- **lg**：1024px - 1280px（小型桌面）
- **xl**：1280px - 1536px（桌面）
- **2xl**：> 1536px（大型桌面）

### 布局适配

**< 768px（移动端）：**
- 单栏布局
- 文件夹和标签改为抽屉
- 底部导航栏
- 书签卡片全宽

**768px - 1024px（平板）：**
- 两栏布局（文件夹 + 书签）
- 标签改为抽屉或底部面板
- 可选三栏布局

**> 1024px（桌面）：**
- 三栏布局（默认）
- 支持调整侧边栏宽度
- 支持折叠侧边栏
