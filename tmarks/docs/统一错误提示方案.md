# 统一错误提示方案

## 1. 设计原则

### 1.1 通知分层

| 类型 | 用途 | 组件 | 特点 |
|------|------|------|------|
| **Toast** | 非阻塞通知 | `useToastStore` | 自动消失，不打断用户 |
| **Dialog** | 阻塞交互 | `useDialogStore` | 需要用户明确响应 |

### 1.2 使用场景

**Toast 适用于：**
- 操作成功反馈（保存成功、删除成功）
- 操作失败提示（网络错误、保存失败）
- 信息提示（已复制到剪贴板）
- 非关键性警告

**Dialog 适用于：**
- 删除确认（不可逆操作）
- 批量操作确认
- 需要用户输入内容
- 重要决策

## 2. 技术方案

### 2.1 新建 `useDialogStore`

**文件位置：** `src/stores/dialogStore.ts`

```typescript
import { create } from 'zustand'

// 确认对话框配置
interface ConfirmOptions {
  title: string
  message: string
  type?: 'info' | 'warning' | 'error'
  confirmText?: string
  cancelText?: string
}

// 输入对话框配置
interface PromptOptions {
  title: string
  message?: string
  placeholder?: string
  defaultValue?: string
  confirmText?: string
  cancelText?: string
}

// 确认对话框状态
interface ConfirmState extends ConfirmOptions {
  isOpen: boolean
  onConfirm: () => void
  onCancel: () => void
}

// 输入对话框状态
interface PromptState extends PromptOptions {
  isOpen: boolean
  onConfirm: (value: string) => void
  onCancel: () => void
}

interface DialogState {
  // 确认对话框
  confirm: ConfirmState | null
  showConfirm: (options: ConfirmOptions) => Promise<boolean>
  
  // 输入对话框
  prompt: PromptState | null
  showPrompt: (options: PromptOptions) => Promise<string | null>
}

export const useDialogStore = create<DialogState>((set) => ({
  // 确认对话框
  confirm: null,
  showConfirm: (options) => {
    return new Promise((resolve) => {
      set({
        confirm: {
          isOpen: true,
          type: 'warning',
          confirmText: '确定',
          cancelText: '取消',
          ...options,
          onConfirm: () => {
            resolve(true)
            set({ confirm: null })
          },
          onCancel: () => {
            resolve(false)
            set({ confirm: null })
          }
        }
      })
    })
  },

  // 输入对话框
  prompt: null,
  showPrompt: (options) => {
    return new Promise((resolve) => {
      set({
        prompt: {
          isOpen: true,
          confirmText: '确定',
          cancelText: '取消',
          ...options,
          onConfirm: (value: string) => {
            resolve(value)
            set({ prompt: null })
          },
          onCancel: () => {
            resolve(null)
            set({ prompt: null })
          }
        }
      })
    })
  }
}))
```

### 2.2 创建全局 `DialogProvider`

**文件位置：** `src/components/common/DialogProvider.tsx`

```typescript
import { useDialogStore } from '@/stores/dialogStore'
import { ConfirmDialog } from './ConfirmDialog'
import { PromptDialog } from './PromptDialog'

export function DialogProvider() {
  const { confirm, prompt } = useDialogStore()

  return (
    <>
      {/* 确认对话框 */}
      {confirm && (
        <ConfirmDialog
          isOpen={confirm.isOpen}
          title={confirm.title}
          message={confirm.message}
          type={confirm.type}
          confirmText={confirm.confirmText}
          cancelText={confirm.cancelText}
          onConfirm={confirm.onConfirm}
          onCancel={confirm.onCancel}
        />
      )}

      {/* 输入对话框 */}
      {prompt && (
        <PromptDialog
          isOpen={prompt.isOpen}
          title={prompt.title}
          message={prompt.message}
          placeholder={prompt.placeholder}
          defaultValue={prompt.defaultValue}
          confirmText={prompt.confirmText}
          cancelText={prompt.cancelText}
          onConfirm={prompt.onConfirm}
          onCancel={prompt.onCancel}
        />
      )}
    </>
  )
}
```

## 3. 使用方式

### 3.1 Toast 通知

```typescript
import { useToastStore } from '@/stores/toastStore'

// 在组件中
const { addToast } = useToastStore()
addToast('success', '保存成功')
addToast('error', '保存失败，请重试')
addToast('info', '已复制到剪贴板')
addToast('warning', '该操作不可撤销')

// 在非组件中（如 hooks、services）
useToastStore.getState().success('保存成功')
useToastStore.getState().error('保存失败，请重试')
```

### 3.2 确认对话框

```typescript

const handleDelete = async () => {

    cancelText: '取消'

    await deleteBookmark(id)

const confirmed = await useDialogStore.getState().showConfirm({

```

### 3.3 输入对话框

```typescript
import { useDialogStore } from '@/stores/dialogStore'

const { showPrompt } = useDialogStore()

const handleImport = async () => {
  const input = await showPrompt({
    title: '导入链接',
    message: '请粘贴要导入的链接，每行一个',
    placeholder: 'https://example.com'
  })
  
  if (input) {
    // 处理输入
    const urls = input.split('\n').filter(Boolean)
    await importUrls(urls)
  }
}
```

## 6. 代码规范

### 6.1 禁止使用

```typescript
// ❌ 禁止
alert('错误信息')
confirm('确定吗？')
prompt('请输入')
window.alert()
window.confirm()
window.prompt()
```

### 6.2 推荐使用

```typescript
// ✅ 推荐
import { useToastStore } from '@/stores/toastStore'
import { useDialogStore } from '@/stores/dialogStore'
import { MESSAGES } from '@/lib/constants/messages'

// 通知
useToastStore.getState().success(MESSAGES.SUCCESS)
useToastStore.getState().error(MESSAGES.FAILED)

// 确认
const confirmed = await useDialogStore.getState().showConfirm({
  title: '确认',
  message: MESSAGES.CONFIRM_DELETE
})

// 输入
const input = await useDialogStore.getState().showPrompt({
  title: '输入',
  placeholder: '请输入...'
})
```
