## 核心设计理念

### 三栏布局结构

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  左侧 (20%)          中间 (55%)                      右侧 (25%)                  │
│  一级文件夹          二级文件夹 + 书签                标签系统（保持现有样式）      │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 设计原则

1. **左侧**：只显示一级文件夹（顶级分类），保持简洁
2. **中间**：显示二级文件夹和书签，以分组形式全部展示
3. **右侧**：完全保留现有标签系统的样式和功能

## 左侧：一级文件夹导航

### 设计特点

```
┌──────────────┐
│ 📁 文件夹     │
│              │
│ [+ 新建]     │
│              │
│ ┌──────────┐ │
│ │📂 全部    │ │ ← 虚拟文件夹（显示所有书签）
│ │  (156)   │ │
│ └──────────┘ │
│              │
│ ┌──────────┐ │
│ │📦 未分类  │ │ ← 虚拟文件夹（folder_id 为 NULL）
│ │  (45)    │ │
│ └──────────┘ │
│              │
│ ┌──────────┐ │
│ │📂 工作    │ │ ← 用户创建的一级文件夹
│ │  (68)    │ │
│ │  ← 选中   │ │
│ └──────────┘ │
│              │
│ ┌──────────┐ │
│ │📂 学习    │ │
│ │  (43)    │ │
│ └──────────┘ │
│              │
│ ┌──────────┐ │
│ │📂 生活    │ │
│ │  (0)     │ │
│ └──────────┘ │
│              │
│ ──────────── │
│              │
│ 🗑️ 回收站     │
│ (5)          │
│              │
│ [折叠 ◀]     │
│              │
└──────────────┘
```

### 功能说明

**虚拟文件夹：**
- **全部**：显示所有书签（不限文件夹）
- **未分类**：显示 folder_id 为 NULL 的书签

**用户文件夹：**
- 只显示一级文件夹（parent_id 为 NULL）
- 不支持展开/折叠（子文件夹在中间区域显示）
- 显示书签总数（包括子文件夹的书签）

**交互：**
- 单击：选中文件夹，中间区域显示该文件夹的内容
- 右键：显示操作菜单（重命名、删除、更改颜色等）
- 拖拽：调整文件夹顺序

**状态：**
- 默认：灰色背景
- 悬停：浅色高亮
- 选中：主色背景，白色文字

✅ **标签搜索**：实时搜索标签
✅ **排序方式**：使用频率、字母序

### 标签筛选逻辑

**独立筛选：**
- 不选文件夹：标签在全局范围筛选

**组合筛选：**
- 选中一级文件夹：标签在该文件夹及其子文件夹范围内筛选
- 选中标签后：中间区域只显示符合条件的书签

**示例：**
```
左侧选中：📂 工作
右侧选中：🏷️ 教程 + 视频

结果：显示"工作"文件夹下，同时有"教程"和"视频"标签的书签
```

## 数据结构

### 文件夹结构

```typescript
// 一级文件夹
interface TopLevelFolder {
  id: string
  user_id: string
  name: string
  parent_id: null              // 一级文件夹没有父级
  level: 1
  position: number             // 排序位置
  color: string | null         // 颜色
  icon: string | null          // 图标
  created_at: string
  updated_at: string
  
  // 统计信息（查询时计算）
  bookmark_count: number       // 总书签数（包括子文件夹）
  direct_bookmark_count: number // 直接书签数
  subfolder_count: number      // 子文件夹数
  
  // 关联数据（查询时加载）
  subfolders?: SecondLevelFolder[]  // 二级文件夹
  direct_bookmarks?: Bookmark[]     // 直接书签
}

// 二级文件夹
interface SecondLevelFolder {
  id: string
  user_id: string
  name: string
  parent_id: string            // 指向一级文件夹
  level: 2
  position: number
  color: string | null
  icon: string | null
  created_at: string
  updated_at: string
  
  // 统计信息
  bookmark_count: number
  
  // 关联数据
  bookmarks: Bookmark[]
}

// 书签
interface Bookmark {
  id: string
  user_id: string
  folder_id: string            // 可以是一级或二级文件夹
  folder_level: 1 | 2          // 所属文件夹层级
  position: number             // 文件夹内排序
  
  // 基本信息
  title: string
  url: string
  description: string | null
  favicon: string | null
  cover_image: string | null
  
  // 标签
  tags: Tag[]
  
  // 状态
  is_pinned: boolean
  is_public: boolean
  is_archived: boolean
  
  // 统计
  click_count: number
  last_clicked_at: string | null
  
  // 时间
  created_at: string
  updated_at: string
}
```
